# mpd
D.C. Traffic Infractions List (searchable)<br />
This is a small Java web application that I developed to slowly integrate, demonstrate and further my training as a developer while trying
to transition back into a software development career.<br />
Code deploys via a .war file
Working ANT and MAVEN scripts to facilitate compiling and deploying
Key tools/technologies:<br/>
Eclipse-Ganymede (IDE)<br />
Java (6) <br />
Tomcat 6.44 (2.5 Servlet API and JSP/EL 2.1)<br />
MySQL 5.5<br />
Key open source libraries used:<br />
BoneCP as database connection pool manager (http://jolbox.com/)<br />
Apache DB Utils (https://commons.apache.org/proper/commons-dbutils/)<br />
Struts 2 (http://struts.apache.org/) along with Struts Tag libraries<br />
Apache Log4J (1.2)<br />
HTML 5/CSS/Javascript (AJAX)<br />
